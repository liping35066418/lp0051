## 1. 产品概述

摄影工作室线上服务平台，为摄影工作室提供套餐管理、预约排期、订单履约全流程数字化运营，同时为用户提供在线选套餐、预约档期、查看拍摄须知、追踪订单状态的一站式体验。
- 解决传统摄影工作室预约混乱、档期冲突、订单流转不透明等痛点
- 面向摄影工作室经营者与终端消费者，提升运营效率与客户体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 后台预设账号 | 套餐管理、摄影师管理、档期管理、订单处理、作品上架、数据统计 |
| 客户 | 手机号注册/登录 | 浏览套餐、在线预约、查看订单、评价服务、浏览作品 |

### 2.2 功能模块

1. **首页**: 品牌展示、精选作品、套餐推荐、工作室介绍
2. **套餐列表页**: 套餐分类筛选（写真/外景/商拍）、价格展示、详情查看
3. **套餐详情页**: 套餐内容、拍摄须知、价格明细、在线预约入口
4. **预约页**: 选择摄影师、选择档期、填写信息、冲突校验
5. **我的订单页**: 订单列表、状态流转、改期/退单操作、成片查看
6. **作品图集页**: 分类浏览、作品详情、点赞收藏
7. **后台管理-套餐管理**: 新增/编辑/上下架套餐
8. **后台管理-摄影师管理**: 摄影师信息维护、档期设置
9. **后台管理-订单管理**: 确认订单、更新状态、处理改期/退单
10. **后台管理-作品管理**: 上架/分类/排序作品图集
11. **后台管理-数据统计**: 访客统计、订单统计、收入统计

## 3. 核心流程

### 预约流程
用户浏览套餐 → 选择套餐 → 选择摄影师 → 选择档期 → 系统校验冲突 → 通过则创建预约订单（待确认）→ 管理员确认 → 拍摄中 → 成片交付 → 完成评价

### 订单状态流转
待确认 → 已确认（拍摄中）→ 成片交付 → 已完成（评价后）

### 改期流程
客户发起改期 → 选择新档期 → 系统校验冲突 → 管理员审批 → 更新档期

### 退单流程
客户发起退单 → 管理员审批 → 退款/拒绝

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 深炭黑(#1a1a2e) + 暖金(#c9a96e)，营造高端摄影工作室质感
- **辅助色**: 象牙白(#f5f0eb)、柔灰(#8a8a8a)
- **按钮风格**: 圆角按钮，主按钮为暖金填充，次要按钮为描边样式
- **字体**: 标题使用 Playfair Display 衬线字体体现优雅感，正文使用 Noto Sans SC 保证中文阅读体验
- **布局风格**: 大图沉浸式首屏，卡片式内容展示，左侧导航后台管理
- **图标风格**: 线性图标，细线条，金色点缀

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero轮播 | 全屏大图，视差滚动，底部渐变遮罩，标语居中白色大字 |
| 首页 | 套餐推荐 | 三列卡片，悬停上浮阴影，金色边框点缀 |
| 首页 | 精选作品 | 瀑布流布局，悬停显示作品信息遮罩层 |
| 套餐列表页 | 分类筛选 | 顶部标签栏，选中态金色下划线 |
| 套餐列表页 | 套餐卡片 | 封面图占上半，下半文字信息，价格金色高亮 |
| 套餐详情页 | 套餐信息 | 左侧大图，右侧信息栏，底部固定预约按钮 |
| 预约页 | 日期选择 | 日历组件，可约日期暖色标记，已约灰色不可选 |
| 预约页 | 摄影师选择 | 横向卡片列表，头像+名称+简介 |
| 我的订单页 | 订单卡片 | 状态标签彩色编码，时间线展示流转记录 |
| 作品图集页 | 作品网格 | 等比网格，悬停放大效果，分类标签叠加 |
| 后台管理 | 侧边导航 | 深色侧边栏，金色活跃项指示器 |
| 后台管理 | 数据统计 | 卡片式数据展示，折线图/柱状图可视化 |

### 4.3 响应式设计

- 桌面优先设计，大屏充分利用空间展示内容
- 平板端自适应两列/单列布局
- 移动端单列布局，底部导航替代侧边栏
- 触屏优化：增大点击区域，滑动操作支持
